# PyGDO Templates

## [GDT_Template](../gdo/core/GDT_Template.py)

I am using "[Templite](../gdo/core/GDT_Template.py)", which is a minimal template engine.
It only features python code with <%%> or <%=%> and treats the rest as strings to output.
